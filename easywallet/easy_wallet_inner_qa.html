  • 一、如何申請/異動
    • 約定連結「悠遊付電支帳戶」服務是什麼？

      本服務提供您以元大銀行臺幣存款帳戶連結您的悠遊付電支帳戶，並於連結成功後，即可提供儲值、付款、提領之服務。

      1. 儲值：您可將元大銀行臺幣存款帳戶款項，儲值至悠遊付電支帳戶。
      2. 付款：您可至店家透過掃碼／出示付款碼方式，由元大銀行臺幣存款帳戶扣款進行消費。
      3. 提領：您可將悠遊付電支帳戶餘額提領至元大銀行臺幣存款帳戶。
    • 約定連結「悠遊付電支帳戶」的申請條件？

      您須符合以下條件：

      1. 本服務限自然人申請。
      2. 持有元大銀行臺幣存款帳戶（含新臺幣活儲存款、新臺幣綜合活儲存款帳戶），或臺幣數位存款帳戶（含第一類、第二類，第三類限以「他行存款帳戶」進行身分驗證者）。
      3. 帳戶無限制交易之狀況。
      4. 悠遊付電支帳戶僅限綁定一組本人的元大銀行存款帳戶。
      5. 需於元大銀行留存正確之手機號碼（作為OTP驗證使用）及電子郵件地址。
    • 如何申請「悠遊付電支帳戶」約定連結服務？

      請透過悠遊付電子支付業者進行申請，步驟如下：

      1. 開啟悠遊付，選擇 「我的→支付工具管理→新增銀行帳戶→元大銀行」。
      2. 輸入連結扣款的元大銀行帳戶資訊。
      3. 以一次性交易密碼（OTP）驗證身分。
      4. 設定成功，即可開始進行付款、儲值、提領等交易！
    • 為什麼約定「悠遊付電支帳戶」連結元大銀行帳戶需留存手機號碼和Email？
      為維護您的交易安全，申請連結元大銀行帳戶時，將傳送一次性密碼（OTP）至您於元大銀行留存的手機號碼以做為身分驗證，另相關服務設定及交易訊息將以電子郵件通知您。
    • 我想申請約定「悠遊付電支帳戶」服務，但還沒有元大銀行存款帳戶怎麼辦？

      您可立即線上申請數位存款帳戶，或持本人身分證、第二證件（如：健保卡、駕照）及印章至元大銀行任一分行申請開立臺幣存款帳戶。

      • 立即開立數位存款帳戶
      • 元大銀行分行據點
    • 可以用他人的存款帳戶約定「悠遊付電支帳戶」嗎？
      本服務僅限以「本人」於元大銀行開立之存款帳戶連結「悠遊付電支帳戶」。
    • 公司戶是否可申請綁定「悠遊付電支帳戶」？
      本服務僅提供元大銀行「個人存款戶」申請（包含外籍人士、未成年人）。
    • 外籍人士是否可申請綁定「悠遊付電支帳戶」？
      外籍人士若符合悠遊付會員註冊規範，且符合元大銀行約定連結「悠遊付電支帳戶」之申請條件，則可透過本人的元大銀行臺幣存款帳戶綁定「悠遊付電支帳戶」。

      註：外籍人士之悠遊付會員註冊流程請詳悠遊付官網查詢
    • 未成年人是否可申請綁定「悠遊付電支帳戶」？
      7歲（含）以上之未成年人，若符合悠遊付會員註冊規範，且符合元大銀行約定連結「悠遊付電支帳戶」之申請條件，並經法定代理人驗證或同意，則可透過本人的元大銀行臺幣存款帳戶綁定「悠遊付電支帳戶」。

      註：未成年人之悠遊付會員註冊流程請詳悠遊付官網查詢
    • 設定連結存款帳戶時，出現「元大銀行OTP超過驗證次數」怎麼辦？
      請洽元大銀行客戶服務專線：（02）2182-1968／0800-688-168，或是任一分行臨櫃辦理。
    • 如何取消「悠遊付電支帳戶」連結的元大銀行存款帳戶？
      請進入悠遊付的「我的」→「支付工具管理」中，點選「元大銀行連結帳戶」，並選擇「刪除」，即可取消連結。
  • 二、交易問題
    • 存款帳戶連結「悠遊付電支帳戶」，有儲值金額上限嗎？
      • 新臺幣活期儲蓄存款帳戶、數位存款帳戶第一類、數位存款帳戶第二類：每筆新臺幣（下同） 5萬元 / 每日10萬元 / 每月20萬元。
      • 數位存款帳戶第三類(限以「他行存款帳戶」進行身分驗證者)：每筆新臺幣（下同） 1萬元 / 每日3萬元 / 每月5萬元。

      註1：以元大銀行帳戶扣款進行儲值及消費之額度合併計算，且綁定不同電子支付業者，其限額依每一位客戶之使用者身分歸戶計算。

      註2：悠遊付另依身分認證等級設有交易款項限額，相關資訊請至悠遊付查詢。

    • 如何自動儲值？

      請先確認您已與元大銀行帳戶綁定，再依下列步驟完成設定：

      1. 選擇「我的」並開啟【自動儲值】功能。
      2. 向右滑動開啟自動儲值功能（向左滑動即關閉自動儲值功能）。
      3. 選擇自動儲值綁定的元大銀行帳戶。
      4. 設定悠遊付帳戶餘額低於多少金額，會啟動自動儲值。
      5. 設定每次自動儲值金額、每日自動儲值上限，並點選儲存。
      6. 進行身分驗證（使用生物辨識或輸入悠遊付設定的登入密碼）

      即可自動儲值成功！

    • 存款帳戶連結「悠遊付電支帳戶」後，當進行消費、儲值或提領後，會有通知嗎？
      當交易完成後，元大銀行會以行動銀行推播、Email或簡訊通知您。
    • 「悠遊付電支帳戶」餘額提領至元大銀行臺幣存款帳戶，會被收手續費嗎？
      若已設定元大銀行臺幣存款帳戶作為連結扣款帳戶，得享提領手續費0元優惠。
    • 「悠遊付電支帳戶」沒有交易紀錄或交易成功通知，綁定的元大帳戶卻被扣款了？
      若對銀行帳務有任何問題，請洽元大銀行客戶服務專線：（02）2182-1968／0800-688-168。
    • 可以查詢「悠遊付電支帳戶」交易明細紀錄嗎？帳務有問題應如何處理？

      您可透過元大銀行之網路銀行或行動銀行，查詢帳戶扣款金額紀錄，亦可於悠遊付「交易紀錄」中查詢交易明細。若有「悠遊付電支帳戶」的帳務問題，請洽悠遊付客服協助：


      悠遊付

      客服電子郵件信箱：service@easycard.com.tw

    • 交易付款完成後，我想退款或退貨應如何處理？
      如您在完成交易後希望返還商品並取消交易，請遵照店家的退款或退貨流程，由店家執行退款功能，將款項退還至您的悠遊付電支帳戶餘額內。
    • 是否所有智慧型手機皆可使用「嗶乘車」或購買1280定期票嗎？
      • 「嗶乘車」裝置限制如下：
        1. 具備NFC功能的Android智慧型手機（目前iOS尚未開放嗶乘車功能）。
        2. 手機系統Operating system版本6.0（含）以上。
        3. 交易時需網路連線，並保持電力充足。
        4. 使用前，請確定手機NFC感應區域。因各款手機裝置NFC感應區域不同，請先向手機品牌廠商查詢該型號之NFC感應區域。
      • 因1280定期票須於「嗶乘車」使用，故僅符合上述條件方可購買。
